LOGS --------------------

Use o módulo logging nativo do Python configurado com um RotatingFileHandler para que seus arquivos de log não cresçam indefinidamente e travem o disco.

Separe os níveis: DEBUG para as respostas brutas da IA, INFO para os passos do sistema, e ERROR com traceback completo para falhas de hardware/microfone.

Testes | pytest + pytest-asyncio  -------------------------------

Verificar se o banco de dados SQLite está salvando e recuperando o histórico corretamente.

Mockar (simular) as respostas do modelo local para testar se as ferramentas (tools) estão sendo disparadas quando palavras-chave são ditas.
